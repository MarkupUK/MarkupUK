Page number : counter(page)

counter(page) is used for generating page numbers[3].

@page {
  @top-right {
    content: "Page " counter(page);
  }
}

[3] counter() was defined by CSS 2.1, but in CSS 3, a preassigned counter for page numbers is introduced in the page context.
